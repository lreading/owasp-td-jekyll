# Security Policy

Vulnerability reports are welcome for supported versions. Please open a [security advisory](https://github.com/lreading/owasp-td-jekyll/security/advisories/new) via GitHub. 

We look forward to working with you to ensure it's resolved and keep our community safe! 

## Supported Versions

This theme was created specifically for [OWASP Threat Dragon's](https://github.com/OWASP/threat-dragon) documentation site - it's unlikely that many others are using this theme. Threat Dragon's docs are regularly maintained and on the latest available for that version. 

For that reason, we are only supporting the latest version.

| Version | Supported          |
| ------- | ------------------ |
| latest   | :white_check_mark: |
| everything prior  | :x:                |
